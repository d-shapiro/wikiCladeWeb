WikiClade

@form(action = routes.HomeController.wikiCladeFormPost()) { @textarea(userForm("inputs")) @inputText(userForm("verbosity")) }

Pass in a list of organisms, get out a cladogram (branching diagram used to illustrate evolutionary relationships) containing them.

Taxonomical relationships are scraped from Wikipedia's infoboxes.

Verbosity affects how many intermediate clades are displayed. 100 means show all clades that any of the input organisms belong to, 0 means only show the input organisms and the branching points.
