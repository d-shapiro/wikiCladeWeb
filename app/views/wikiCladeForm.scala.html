WikiClade

@form(action = routes.HomeController.wikiCladeFormPost()) { @textarea(userForm("inputs")) @inputText(userForm("verbosity")) }

This tool scrapes Wikipedia to generate a cladogram (branching diagram used to illustrate evolutionary relationships) linking the input organisms. Verbosity affects how many intermediate clades are displayed. At 0, only the target organisms and branching points will be displayed. At 100, all known intermediate clades will be displayed.
